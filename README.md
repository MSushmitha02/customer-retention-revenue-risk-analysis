# Customer Retention & Revenue Risk Analysis

## Problem
Identify customers showing declining engagement and estimate potential revenue at risk using transaction-level data.

## Dataset
Online Retail II dataset (transaction-level e-commerce data).

## Approach
- Converted transaction data into customer-level behavioral metrics
- Calculated recency, frequency, and revenue contribution
- Segmented customers into Loyal, At-Risk, and Dormant groups

## Key Insights
- A subset of high-value customers shows reduced recent activity
- A significant portion of revenue is exposed to churn risk

## Tools
Python, pandas, numpy, matplotlib, seaborn
